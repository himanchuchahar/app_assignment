# App-Assignment
Challenge Application for Car portal


Installation steps:
npm install will install all the dependencies

After successful installation type localhost:3000 in browser


## Feature Implementation:
 - Implement function based component along with react hooks for better performance
 - Implement redux for state management along with thunk middlware to dispatch async actions
 - Implement test-cases using jest and enzyme
 - Implement global + component styling using materialUI 
 - Implement Axios to get data from server
 - Implement add/remove saved cars using localStorage
 - Implement responsive design for mobile, tablet and desktop


## Below are the libraries used
 - React-router: to handle routing on client side
 - React-redux: for state management
 - Redux-thunk: middleware for actions
 - Material UI: for styling components
 - Jest and enzyme: for test-cases
 - axios: for server interaction
